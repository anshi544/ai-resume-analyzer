# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit.  
This project analyzes uploaded resumes in PDF format, detects skills, calculates a resume score, and suggests improvements.

## Features

- Upload PDF Resume
- Skill Detection
- Resume Score Analysis
- Missing Skills Suggestions
- Beginner-Friendly UI

## Tech Stack

- Python
- Streamlit
- PyPDF2
